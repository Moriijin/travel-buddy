# 搭伴游 - UI/UX 设计指南

## 📋 设计原则

基于业界最佳实践，我们遵循以下设计原则：

### 1. 视觉层次 (Visual Hierarchy)
- **尺寸**：重要元素更大更醒目
- **色彩**：使用明亮色彩吸引注意力（如按钮用紫色渐变）
- **对比度**：关键操作使用高对比度
- **间距**：留白分隔和分组元素
- **字体**：粗细和样式强调重点

### 2. 一致性 (Consistency)
- 统一的组件行为
- 一致的术语和标签
- 统一的视觉风格和间距
- 一致的交互模式

### 3. 简洁性 (Simplicity)
- 移除不必要的元素
- 聚焦核心功能
- 渐进式展示（需要时显示更多信息）
- 简洁明了的文案
- 减少认知负荷

### 4. 反馈与响应 (Feedback & Response)
- 用户操作的即时视觉反馈
- 异步操作的加载状态
- 成功/错误提示
- 长时间操作的进度指示
- 微交互带来愉悦感

### 5. 错误预防与恢复 (Error Prevention & Recovery)
- 内联表单验证
- 破坏性操作需要确认
- 清晰的错误提示
- 提供修复建议
- 允许撤销操作

## 🎨 色彩系统

### 主色系（马卡龙色系）
| 颜色 | 色值 | 使用场景 |
|------|------|----------|
| 薄荷绿 | `#B8E6D0` | 成功状态、积极情绪 |
| 淡紫色 | `#C5B8E6` | 主品牌色、导航 |
| 蜜桃粉 | `#FFB8A3` | 警告状态、温暖元素 |
| 奶油黄 | `#FFE0A3` | 提示、强调 |
| 天空蓝 | `#A3CFFF` | 信息、辅助色 |
| 玫瑰粉 | `#FFB8C6` | 女性标识、社交元素 |
| 柠檬黄 | `#FFF3A3` | 高亮、特殊标记 |

### 渐变效果
- **主渐变**: `linear-gradient(135deg, #C5B8E6, #A3CFFF)`
- **暖渐变**: `linear-gradient(135deg, #FFB8A3, #FFE0A3)`
- **清新渐变**: `linear-gradient(135deg, #B8E6D0, #A3CFFF)`

### 文字色彩
| 类型 | 色值 | 用途 |
|------|------|------|
| 主文字 | `#3D3557` | 标题、重要信息 |
| 副文字 | `#7B7399` | 描述、辅助信息 |
| 弱化文字 | `#B8B0CC` | 时间戳、标签 |
| 背景色 | `#F8F5FF` | 页面背景 |
| 卡片背景 | `#FFFFFF` | 组件背景 |

## 🔤 字体规范

- **字体家族**: 'PingFang SC', 'Microsoft YaHei', sans-serif
- **标题**: 18-24px, font-weight: 600-700
- **正文**: 13-15px, font-weight: 400-500
- **辅助文字**: 10-12px, font-weight: 400

## 📐 间距系统

使用 4px 基础单位：
- **组件内间距**: 8px, 12px, 16px, 20px, 24px
- **组件间距**: 12px, 16px, 20px
- **页面边距**: 16px

## 🧱 组件规范

### 1. 卡片 (Card)
- **圆角**: 20-24px
- **阴影**: `0 4px 20px rgba(61, 53, 87, 0.08)`
- **悬停阴影**: `0 6px 24px rgba(61, 53, 87, 0.12)`
- **装饰条**: 顶部 4px 渐变色条

### 2. 按钮 (Button)
- **主按钮**: 紫色渐变背景，白色文字
- **次按钮**: 虚线边框，紫色文字
- **危险按钮**: 红色背景
- **图标按钮**: 44x44px 最小触控区域
- **状态**: 默认、悬停、按下、禁用、加载

### 3. 标签 (Tag)
- **圆角**: 16px
- **内边距**: 4px 12px
- **字体大小**: 11px
- **背景**: 半透明彩色背景

### 4. 输入框 (Input)
- **圆角**: 16px
- **边框**: 2px solid var(--lavender)
- **焦点状态**: 边框变深
- **错误状态**: 红色边框
- **成功状态**: 绿色边框

### 5. 筛选器 (Filter)
- **样式**: 胶囊形状
- **未选中**: 半透明紫色背景
- **选中**: 紫色渐变背景，白色文字
- **圆角**: 20px

## 🎭 动画规范

### 过渡效果
- **时长**: 200-300ms
- **缓动**: ease / ease-in-out
- **使用场景**: hover、click、页面切换

### 悬停效果
- **卡片**: `transform: translateY(-2px)`
- **按钮**: 背景色变化
- **列表项**: 轻微上浮

## 📱 响应式设计

### 移动端优先
- **最小宽度**: 320px
- **触控区域**: 最小 44x44px
- **字体大小**: 不小于 12px
- **行高**: 1.4-1.6

### 布局模式
- **单列布局**: 主要内容区域
- **网格布局**: 2 列或 3 列数据展示
- **弹性布局**: Tab、标签等

## ♿ 无障碍设计

### WCAG 2.1 AA 标准
- **对比度**: 文字至少 4.5:1
- **键盘导航**: 所有交互元素可用键盘操作
- **焦点指示**: 清晰的焦点样式
- **屏幕阅读器**: 语义化 HTML、ARIA 标签
- **图片替代**: 所有图片有 alt 描述

## 📊 数据可视化

### 图表规范
- **柱状图**: 使用主色系渐变
- **进度条**: 圆角 4px
- **统计卡片**: 左/顶部彩色条装饰

### 颜色编码
- **成功**: 薄荷绿 `#B8E6D0`
- **进行中**: 淡紫色 `#C5B8E6`
- **警告**: 蜜桃粉 `#FFB8A3`
- **信息**: 天空蓝 `#A3CFFF`

## 🎯 页面设计要点

### 首页 (搭子推荐)
- 欢迎回来区域
- 心情选择器（情感化设计）
- 数据统计卡片
- 筛选栏
- 搭子推荐卡片列表

### 发现页
- 搜索栏
- 多级筛选（距离、兴趣）
- 热门话题卡片
- 性格测试（增加互动性）

### 行程页
- Tab 切换（我的/发现）
- 行程卡片（路线、时间、费用）
- 报名按钮

### 消息页
- 聊天列表
- 未读消息标记
- 最新消息预览

### 我的
- 用户信息卡片
- 信用分展示
- 心情日历
- 月度总结
- 功能菜单

## 🚀 性能优化

### 图片优化
- 使用 WebP 格式
- 懒加载
- 响应式图片

### 代码优化
- CSS 变量减少重复
- 最小化重绘重排
- 使用 transform 和 opacity 动画

### 加载状态
- 骨架屏
- 渐进式加载
- 加载动画

---

**设计理念**: 以用户为中心，创造温暖、友好、易用的旅行社交体验。通过马卡龙色系和卡通元素，营造轻松愉快的氛围，让找搭子旅行变得简单有趣。
