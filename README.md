# 搭伴游 (Travel Buddy)

一款面向 18-35 岁年轻人的旅行 companion 匹配平台微信小程序。

## 项目简介

"搭伴游"帮助旅行者找到志同道合的旅伴，通过智能匹配算法，让旅途不再孤单。

### 核心功能

- **首页推荐** - 发现有趣的旅行者和活动
- **智能匹配** - 基于旅行偏好、时间、地点的匹配算法
- **旅行计划** - 创建和分享旅行行程
- **即时聊天** - 与匹配成功的旅伴交流
- **个人资料** - 管理个人旅行档案和偏好

## 技术栈

- 微信小程序原生开发
- 微信云开发（可选）
- 微信 UI 组件库

## 项目结构

```
travel-buddy/
├── app.js                  # 小程序入口
├── app.json                # 全局配置
├── app.wxss                # 全局样式
├── project.config.json     # 开发者工具配置
├── pages/                  # 页面目录
│   ├── index/              # 首页
│   ├── profile/            # 个人资料
│   ├── match/              # 匹配页面
│   ├── chat/               # 聊天页面
│   └── travel-plan/        # 旅行计划
├── utils/                  # 工具函数
└── assets/                 # 静态资源
```

## 快速开始

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入本项目目录
3. 点击"编译"即可预览

## 开发指南

### 页面开发规范

- 每个页面包含 4 个文件：`.js`, `.json`, `.wxml`, `.wxss`
- 遵循微信小程序官方编码规范
- 使用 rpx 作为尺寸单位（750rpx = 屏幕宽度）

### 样式规范

- 主色调：薰衣草紫 (#B5A6E6)
- 渐变：紫色到粉色
- 圆角卡片设计
- 年轻化、活泼的视觉风格
